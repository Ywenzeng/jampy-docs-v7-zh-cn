========================
Jam.py V7 文档
========================

简介
==============

欢迎使用 Jam.py！如果您是 Jam.py 或无代码、低代码或多代码 Web 应用程序开发的新手，这里可以找到关于 Jam.py V7 的文档。

与 Jam.py V5 最大的区别在于增加了 :doc:`路由 </admin/routing>` 支持和采用了 Bootstrap 5，从而实现对移动设备的现代化支持（详见 :doc:`版本 7 </releases/version_7/index>`）。

此外，用于桌面或移动设备的完整界面都采用无代码方式驱动。
只需为任何设备选择所需的功能，即可开始使用。

.. admonition:: 目标

    安装 Python 和 Jam.py，选择数据库和 Web 服务器，并做出应用程序设计决策。

.. admonition:: 读者对象

    Web 开发爱好者或开发人员，对 Web 开发或部署经验有限或无经验。

.. admonition:: 前提条件

    建议具备一些 Python 和 JavaScript 知识。需要掌握命令行提示符的基本知识，并能够输入命令。

文档结构概览
==================================

以下是文档结构的概览，以帮助您找到特定内容的所在位置：

:doc:`入门指南 <intro/index>` 主题描述了如何安装框架、创建新项目、逐步开发 Web 应用程序并部署它。

:doc:`编程指南 <programming/index>`
在较高层次上讨论关键主题和概念，并提供有用的背景信息和解释。

:doc:`业务应用程序构建器 <admin/index>` 是对用于应用程序开发和数据库管理的应用程序构建器的详细描述。

:doc:`类参考指南 <refs/index>` 包含 Jam.py 类 API 的技术参考。

:doc:`常见问题 <faq/index>` 主题涵盖了最常被问到的问题。

:doc:`操作指南 <how_to/index>` 包含一些代码示例，对于快速完成常见任务可能很有用。

请访问 :doc:`目录 <contents>`，或者查看

`Jam.py 应用程序设计技巧`_，以获取关于如何构建应用程序或从 MS Access 迁移的详细步骤。

.. _`Jam.py 应用程序设计技巧`: https://jampy-application-design-tips.readthedocs.io/

要将本文档下载为单个 PDF 文件，请访问：


https://jampy-docs-v7.readthedocs.io/_/downloads/zh-cn/latest/pdf/
