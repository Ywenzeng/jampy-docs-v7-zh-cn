============
演示项目
============

该框架包含一个功能完整的演示应用程序，展示了框架中使用的编程技术。

演示程序位于从 Github 下载的 Jam.py 软件包的 demo 文件夹中。此外，还提供了一个适用于 Windows x64 的独立版或便携式应用程序，可从 `此处 <https://github.com/platipusica/jampy-exe/releases/download/Pump/jampy_win_64.exe>`_ 获取。

该便携式应用程序仅依赖 LibreOffice 生成报告，无需其他配置。只需运行它，并按照以下步骤在浏览器中访问即可。

要启动演示应用程序，请进入 demo 文件夹并运行 *server.py* 脚本。

.. code-block:: console

    $ ./server.py


打开 Web 浏览器，在地址栏中输入：

.. code-block:: console

	127.0.0.1:8080


要查看
:doc:`应用程序构建器 </admin/index>`，
请在浏览器中打开一个新页面并输入：

.. code-block:: console

	127.0.0.1:8080/builder.html
